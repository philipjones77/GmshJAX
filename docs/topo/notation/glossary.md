# Topo Glossary

Key Topo terms:

- domain: the initialized geometric/problem setup used to start a workflow
- workflow run: the structured result and artifact bundle returned by a mode workflow
- topology: the fixed-shape connectivity substrate represented by `MeshTopology`
- artifact: exported mesh, metrics, history, snapshot, STL, or visualization payload output
- bridge: RF77-oriented payload surface built from a result or runtime state
- family: element family such as `tri`, `quad`, or `tet`
