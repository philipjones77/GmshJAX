# Topo Runtime Objects

Primary Topo runtime objects include:

- `MeshTopology`
- `MeshModel` and `MeshState`
- `Mode1Domain` through `Mode5Domain`
- `Mode1WorkflowRun` through `Mode5WorkflowRun`
- `GmshElementBlock`
- `TopoSnapshot`

These objects carry the practical Topo workflow contract for initialization, optimization, diagnostics, export, and bridge generation.
