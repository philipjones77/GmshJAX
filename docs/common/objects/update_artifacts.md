# Common Update Artifacts

When the shared layer changes, the usual artifacts that should be refreshed are:

- docs under `docs/common/`
- shared inventories under `docs/common/reports/`
- protocol or standards docs under `docs/common/specs/` and `docs/common/standards/`
- tests under `tests/common/`
- benchmarks under `benchmarks/common/`
- examples under `examples/common/`

If the change affects a backend adapter, the corresponding backend docs and tests should also be refreshed.
