# Milestone M2: Static Generator

This milestone note tracks the shared-layer work needed to keep static mesh generation and shared runtime exports coherent across the repository.

M2 common-track targets:

- stable shared protocol documents under `docs/common/specs/`
- one common movement-transform definition
- one common diagnostics and IO story
- one NumPy mesh runtime for RF77-facing array workflows
- mirrored documentation structure so the common layer can be audited like the backend tracks

Implementation details for the current shared runtime live under `implementation/`.
